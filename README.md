# Personal Portfolio Website

A clean, modern personal website template for students, researchers, and developers to showcase projects, experience, and professional work.

---

## Overview

This template is designed to present technical work clearly and professionally. It supports multi-page layouts, expandable sections, and embedded previews for documents, images, and videos.

---

## Features

- Minimal, modern design
- Multi-page layout (About, Projects, Experience, Resume)
- Expandable project and experience sections
- Embedded PDF, image, and video previews
- Responsive and lightweight (no frameworks required)

---

## Usage

Open `index.html` locally to preview the site.  
Deploy using any static hosting service such as GitHub Pages.

---

## Design Philosophy

- Content-first presentation
- Clean visual hierarchy
- Technical depth without clutter
- Visuals used to support explanations

---

## Intended Use

Suitable for:
- Academic and research portfolios
- Technical project showcases
- Internship and graduate applications
- Personal professional websites
